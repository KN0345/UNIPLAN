# UniPlan v0.9.18 資料架構

本版維持資料域分離，避免學生端頁面直接混用不同來源的資料。

```txt
frontend/src/data/
├─ courses/      課程查詢資料與課程顯示欄位定義
├─ graduation/   畢業學分規則與進度摘要
├─ programs/     學程目錄、群組規則、進度判定
├─ schedule/     課表與暫存區資料形狀
└─ users/        使用者與帳號資料形狀
```

## 學程資料原則

學生端只看「可選課程方塊」，不顯示規則術語。規則只存在資料層，用來控制：

- 必修固定課程
- 單選群組
- 多選群組
- 已修/已選/不可選狀態
- 加入暫存區前的重複檢查
