# UniPlan v1.0 Alpha Public Ready

本版目標是先公開測試手機端與排課流程，不綁正式帳號，不串接學校教務系統。

## 本版定位

- 預設進入本機模式
- 不要求註冊或登入
- 使用 localStorage 保存使用者排課資料
- 保留未來帳號系統與雲端同步接口
- 新增學生端意見箱入口

## 不做項目

- 不做淡江單一登入
- 不做學校教務系統爬蟲
- 不要求學生輸入真實密碼
- 不收集敏感資料

## 測試重點

1. 手機端課程卡片是否清楚
2. 暫存區與排課流程是否順
3. 學程方塊選課器是否好懂
4. 畢業學分頁是否足夠簡潔
5. 課程資料錯誤能否透過意見箱回報
