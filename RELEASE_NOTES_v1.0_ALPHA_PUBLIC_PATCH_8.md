# UniPlan v1.0 Alpha Public Patch 8

## 修正重點

1. 修正「已被當」課程邏輯：
   - 已被當課程會保留在原課表上作為紀錄。
   - 已被當課程不計入畢業學分與學程完成進度。
   - 已被當課程不再阻擋學生於後續學期重新加入同名課程。

2. 修正學程進度判定：
   - 學程進度只採計已排課與暫存課。
   - 收藏不再視為已完成或已規劃。
   - 已被當課程不再讓學程誤判通過。

3. 修正課程查詢卡片視覺：
   - 解決左上狀態點、開課序號、課名互相壓到的問題。
   - 學分維持左下小標籤。
   - 降低塑膠反光層高度與透明度，避免文字被干擾。

4. 修正意見箱版面：
   - 最近回報獨立成右側欄。
   - 最近回報對齊頁面標題高度。
   - 桌面版避免整頁滾動，右側列表自行滾動。

5. 修正管理端統計：
   - 移除重複的待處理與系統狀態資訊。
   - 保留總覽與異常中心作為主要資料管理視角。

6. 修正彈窗關閉按鈕：
   - 管理端與學程課程選擇彈窗的關閉按鈕固定在 header 右側，不再漂移。

## 測試

- `npm run build` 通過。
