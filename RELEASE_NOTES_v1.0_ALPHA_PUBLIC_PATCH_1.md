# UniPlan v1.0 Alpha Public Patch 1

## Scope
This patch focuses on public alpha usability feedback.

## Changes
- Course cards now expand instead of compressing text.
- Graduation page restores minimal category progress details.
- Program chooser sticky header is simplified to keep only the add-to-candidate action and progress count.
- Feedback categories are simplified to: course/program data, account, display, function.
- Student feedback page no longer shows prior local reports.
- Admin console now has three tabs: feedback reports, account management, data statistics.
- Admin feedback cards can be opened to view details and update status.
- Rejected or processed feedback is removed from the active feedback list.
- Account management is limited to the current admin account 414730209 by default.
- Roles are simplified to admin and student.
- Data statistics now shows clearer cards, issue categories, and duplicate course codes with course names.

## Build
- Frontend build passed.
