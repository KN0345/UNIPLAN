# UniPlan v1.0 Beta Candidate Features

## 本版重點

本版依照 Beta Candidate 設計，加入可先設計與實作的公開測試功能，暫不處理手機版與正式帳號系統。

## 已完成

1. 課程資訊頁補強
   - 顯示教師、時間、教室、開課序號、開課單位、必/選修、課程簡介。
   - 顯示此課程出現在哪些學程與群組中。

2. 課程標籤投票
   - 提供 12 個固定標籤：作業少、作業多、給分高、給分低、點名多、點名少、考試多、考試少、報告多、報告少、分組多、分組少。
   - 使用者點擊標籤即可投票，再次點擊可取消。
   - 標籤投票改為依課程識別，不再依教師名稱。

3. 課程搜尋標籤排序
   - 課程搜尋篩選列加入「標籤」。
   - 選擇標籤後，課程會依該標籤得票數排序。
   - 不隱藏 0 票課程，避免使用者以為資料消失。

4. 我的方案
   - 「快照紀錄」改為「我的方案」。
   - 不提供預設方案，畫面保持空白等待學生自行建立。
   - 可新增方案、套用方案、匯入/匯出文字備份。

5. 資料中心
   - 管理端「資料統計」改為「資料中心」。
   - 強調資料缺失、重複、規則未完整皆集中於異常中心。
   - 找不到或尚未釋出的資料可先保留統計，後續補充。

6. 新手導引簡化
   - 僅保留找課、暫存、放入課表三步。
   - 不再引導學程。

## 暫不處理

- 通知中心主功能：UniPlan 目前偏向一次性工具，通知主要保留給未來 Widget。
- 手機版深度調整：目前以電腦版公開測試為主。
- 正式帳號系統：仍先採本機模式。

## Build

- npm run build 通過。
