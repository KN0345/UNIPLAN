# UniPlan v1.0 Alpha Public Patch 9

## 修正重點

- 修正課程查詢卡片的文字、開課序號與學分標籤重疊問題。
- 課程卡片重新保留清楚的層級：左上狀態點、右上開課序號、左下學分、右下操作按鈕。
- 學程進度改為精準課名比對，避免「資訊安全」誤判為「資訊安全管理」。
- 學程班別選擇也改用精準課名比對，避免相似名稱互相影響。
- 課程資訊頁新增「對應學程」，可查看該課出現在哪些學程與群組。
- 意見箱與最近回報切成左右獨立區塊，最近回報從右側頂端開始對齊並可拉長。
- Build 測試通過。

## 注意事項

- 對應學程目前以課名精準比對為主，若未來有同義課、改名課，建議新增別名對照表。
- 最近回報在手機版會自然改成上下排列。
