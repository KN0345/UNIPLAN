# UniPlan v1.0 Beta Candidate Final UX

## 版本定位

本版為小規模公開測試前的體驗收斂版，重點放在課程心得、標籤排序、我的方案、課程比較與新手導引。

## 更新內容

1. 課程資訊頁保留並強化課程心得與標籤投票。
2. 搜尋頁支援標籤篩選，並依該標籤得票排序。
3. 新增課程比較流程，可選 2–4 門課並排比較。
4. 比較欄位包含課名、教師、時間、學分、必/選修、開課單位、開課序號、最高票標籤。
5. 我的方案頁改為學生自建方案，不提供預設方案。
6. 我的方案只保留建立、替換、刪除三個主操作。
7. 新手導引維持找課、暫存、放入課表三步，不帶學程。
8. build 測試通過。

## 已知限制

1. 課程心得、標籤投票、方案目前仍為本機儲存。
2. 課程比較前期會受標籤票數不足影響。
3. 帳號系統尚未雲端化。
