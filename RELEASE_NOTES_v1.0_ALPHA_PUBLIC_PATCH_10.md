# UniPlan v1.0 Alpha Public Patch 10

## 修正內容

- 更新「商業財經學分學程」資料：核心12學分、財金選修6學分、國企選修6學分，選修同系超過6學分不再推進進度。
- 更新「財經法律學分學程」資料：依109年3月修正版拆成四個主要群組。
- 新增學程規則 `MAX_CREDIT`，支援「最多採計X學分」進度上限。
- 學程選課頁支援巢狀群組展開，避免子群組課程不顯示。
- 課程資訊頁對應學程支援巢狀群組搜尋。
- 課程查詢卡片改成穩定霧面深藍卡片，避免序號、課名、教師與學分重疊。
- 學程卡片套用同款霧面深藍樣式，並保留已選、已修、必修與不可選狀態色。
- 意見箱表單與最近回報維持獨立雙欄，右側最近回報可拉長並固定在頁面右側。

## Build

- `npm run build` 通過。
