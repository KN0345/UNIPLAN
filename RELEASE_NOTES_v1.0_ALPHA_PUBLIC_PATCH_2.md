# UniPlan v1.0 Alpha Public Patch 2

## 本版重點

- 課程查詢卡片調整：左下顯示學分，右上顯示開課序號。
- 畢業學分細項進度條改為不同顏色，總進度維持藍綠漸層。
- 學程選課器改成直向課程列。
- 學程課程列右側加入小圓資訊按鈕，可查看對應課程並開啟課程資訊。
- 學程「加入暫存區」按鈕不再跟隨內容捲動。
- 意見箱加入最近回報，避免重複回報同一問題。
- 管理端意見內容改為彈出視窗。
- 管理端資料統計整合為缺失/異常總覽，詳細資料改為彈出視窗。
- 重複課號只在同一課號對應不同課名時列為異常；同課不同班別不列入問題。
- 管理端帳號仍只保留唯一最高權限管理員：414730209。
