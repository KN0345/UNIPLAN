# UniPlan v1.0 Alpha Public Patch 4

## 本版重點

- 課程卡片學分改成左下小標籤。
- 畢業學分細項改成與總進度相同的橫向進度條樣式。
- 意見箱右側最近回報縮窄、加高並與表單頂部對齊。
- 學程名稱移除「淡江大學」前綴。
- 隱藏目前缺完整資料的「高齡健康服務人才培力」與「食農教育人才培力」學程。
- 學程課程改用課名比對本學期課程資料。
- 多班別課程改成全域彈窗，避免資訊視窗被卡片容器切掉。
- 學程群組加入進度條與門檻提醒；達標後仍可繼續選課。
- 企業訓練與數位學習學程會依學生系所套用企管/教科修課組合。
- 已修/已規劃課程綠色狀態加深，提升辨識度。
- 管理端缺失資料加入學年資訊。
- 重複課號繼續排除「同課名不同班別」情況。
- package scripts 改為直接呼叫 Vite，降低不同環境中 .bin 解析錯誤。

## Build

- `npm run build` 通過。
