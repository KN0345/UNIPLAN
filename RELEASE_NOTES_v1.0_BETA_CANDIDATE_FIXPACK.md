# UniPlan v1.0 Beta Candidate Fix Pack

## 修正項目

1. 學程課程查找改為「去班別後精準同名」：可找到「教育科技概論 (A班)」等實際班別，同時避免「資訊安全」誤連動「資訊安全管理」。
2. 學程卡片狀態色重整：加入暫存或已排入規劃維持藍色，只有真正已修過才顯示綠色。
3. 課程搜尋卡片縮小，並修正序號、課名、教師、學分、按鈕重疊問題。
4. 比較按鈕改為與「加入暫存 / 收藏」同列的小按鈕，放在加入暫存左側。
5. 課程比較頁新增比較摘要、標籤票數與差異觀察，不再只是資訊並列。
6. 課表時間解析支援「四 / 6,7」、「一 / 3,4 五 / 2」等格式，避免加入課表後顯示未列出時間。
7. 儲存快照改為開啟「我的方案」彈窗，可建立、替換、刪除、套用方案。
8. 匯出課表 PNG 改用 canvas 匯出流程，較能支援背景圖片。
9. 課程資訊頁對應學程比對同步支援去班別後精準同名。

## 驗證

- `npm run build` 通過。
