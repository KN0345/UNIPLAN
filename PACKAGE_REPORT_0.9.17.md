# UniPlan v0.9.17 打包紀錄

## 本版目標

本版聚焦於資料一致性與手機版可用性，不新增首頁功能，不重新啟用 CSV 匯入主流程。

## 資料夾架構整理

主要維持以下資料分層：

```txt
uni/
├─ src/                         # 後端 FastAPI 主程式
├─ data/                        # SQLite 與畢業規則資料
├─ raw_data/                    # 原始課程資料
├─ frontend/
│  ├─ src/
│  │  ├─ App.jsx                # 主要前端整合入口
│  │  ├─ api.js                 # API 呼叫
│  │  ├─ style.css              # 介面樣式
│  │  ├─ pages/
│  │  │  └─ ProgramsPage.jsx    # 學程方塊選課器
│  │  ├─ components/
│  │  │  └─ admin/              # 管理端輕量頁面
│  │  └─ data/
│  │     ├─ courses/            # 課程資料說明
│  │     ├─ graduation/         # 畢業規則資料說明
│  │     ├─ programs/           # 學程資料與規則
│  │     ├─ schedule/           # 課表狀態資料說明
│  │     └─ users/              # 使用者資料說明
│  └─ dist/                     # build 輸出
```

## 本版修改

1. 課程卡片手機版降噪：保留教師姓名、課名、學分、時間，隱藏推薦標籤與課程來源角標。
2. 學程頁改為更接近方塊選課器：課程方塊只顯示課名，顏色承擔狀態意義。
3. 學程方塊顏色邏輯：必修紅色、已選/已修綠色、不可選灰色、一般課程跟隨主題色。
4. 管理頁面維持統計、缺失提醒、帳號管理、意見箱，不顯示規則編輯器或技術文字。
5. 保留首頁與主要排課流程，避免破壞既有使用習慣。
