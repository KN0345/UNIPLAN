# UniPlan v1.0 Alpha 邊界

## 本版定位
UniPlan 是學生端排課與修課模擬工具，不是教務行政系統。

## 明確包含
- 課程搜尋與收藏
- 暫存課表與八學期規劃
- 衝堂檢查
- 畢業學分進度條
- 學程方塊選課器
- 一鍵加入暫存區
- 管理模式：統計、缺失提醒、帳號管理、意見箱
- 本地帳號與資料保存

## 明確不包含
- 教務系統爬蟲
- 學校 API 串接
- 淡江 SSO / OAuth 正式登入
- 學程申請資格判斷
- 學程申請時間提醒
- 證書核發或審核流程
- 正式選課送出

## 設計原則
學生端只顯示完成選課規劃必要資訊；規則、技術代碼與行政文字保留在資料層或管理端。
