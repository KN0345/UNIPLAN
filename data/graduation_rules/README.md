# Graduation Rules Databases

This folder stores structured graduation-rule databases for UniPlan.

Databases:
- graduation_rules_113.db
- graduation_rules_114.db

Main Rule Types:
- Credit requirements
- Elective / required elective groups
- Prerequisite rules
- Course substitution rules
- Credit exclusion rules
- Department merge / rename / closure rules
